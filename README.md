# 🏡 House Price Prediction using Machine Learning and Streamlit

This project predicts house prices based on various features using a machine learning model and provides a user-friendly interface with **Streamlit**. The model is trained on housing data and can be used to estimate the selling price of a house based on user inputs.

---

## 📌 Features

- Predicts house prices using features like area, location, quality, number of rooms, etc.
- Trained using **Random Forest Regressor**.
- Interactive UI built with **Streamlit**.
- Data preprocessing and feature encoding handled programmatically.
- Deployment-ready with saved model and feature set using **Joblib**.

---

## 🚀 Tech Stack

- **Python**
- **scikit-learn**
- **pandas**
- **numpy**
- **matplotlib**
- **Streamlit**
- **joblib**

---

## 🧠 Machine Learning Workflow

1. **Data Cleaning** – Handle missing values, encode categorical variables.
2. **Feature Engineering** – Select relevant features for training.
3. **Model Training** – Use Random Forest Regressor for prediction.
4. **Evaluation** – Evaluate model using R² Score.
5. **Deployment** – Streamlit-based frontend to collect user input and show prediction.
